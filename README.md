# ClearMark

Version: `v1.0.0`

ClearMark is a personal icon set for my traffic dispatch and proxy profile configuration.

## Style

ClearMark is the finalized icon style for the Aurora Marks collection: clean, readable, transparent PNG marks designed for personal traffic dispatch and proxy profile setups.

ClearMark uses a crisp transparent presentation:

- recognizable service and policy marks
- official-proportion country and region flags
- small two-letter country and region codes placed outside the flag at the lower-right edge
- country and region flags without external card-style outlines
- near-white flag areas tuned for visibility on light app backgrounds
- simple map icons for regional routing groups
- native 1024px transparent PNG files with clean vector-style edges

## Format

- Size: `1024x1024`
- Format: optimized PNG
- Content scale: visually balanced per icon type
- Background: transparent canvas without global shadow or glass haze
- Intended use: personal Surge, Clash, Stash, and related network profile setups

## Naming

- Use readable service names, for example `Proxy.png`, `PayPal.png`, `WhatsApp.png`, `Twitter.png`, `TV.png`
- Use two-letter uppercase country and region codes, for example `HK.png`, `JP.png`, `RU.png`, `IN.png`, `IE.png`, `TR.png`, `BR.png`, `SE.png`
